Nombre del proyecto :Juego del ahorcado 
Integrantes :Marcos Benavides
Objetivo del sistema :El objetivo principal del sistema es desarrollar un software interactivo basado en consola que automatice de forma eficiente el juego clásico "El Ahorcado", utilizando el lenguaje Python.
El programa busca ofrecer una herramienta lúdica y accesible para la estimulación cognitiva y el fortalecimiento del vocabulario, demostrando de manera práctica la aplicación de las estructuras de control, la 
modularidad y las técnicas de validación de datos adquiridas en la materia de Lógica de Programación.
Descripción de funcionalidades 
El juego está diseñado de forma modular bajo la consola de comandos e incluye las siguientes características:
Selección Aleatoria: Escoge automáticamente una palabra distinta de un banco de datos en cada partida.
Validación de Entradas: Controla de forma estricta que el usuario ingrese solo letras válidas (evita números, caracteres especiales o letras repetidas).
Actualización en Tiempo Real: Muestra dinámicamente el progreso de la palabra oculta (_ _ _ _ _) y las letras utilizadas.
Gestión de Intentos: Cuenta de forma exacta las vidas restantes y descuenta puntos/intentos ante cada fallo.
Control de Estado: Determina automáticamente el fin de la partida, desplegando pantallas personalizadas en caso de victoria o derrota.
